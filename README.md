# AwsCodePipeline
Utilização do ECR,  CodeCommit,  CodeBuild, CodeDeploy.
Implantação automática do app em HTML com CI/CD CodePipeline. Sobe em uma EC2 para testes e já manda um SNS para aprovação e subir em produção.
